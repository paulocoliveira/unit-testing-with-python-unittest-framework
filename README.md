## Installation
As a prerequisite for this code, you need:
1. Download and Install Python
2. Install unittest, and Selenium WebDriver libraries
```bash
pip install -r requirements.txt
```
3. Download and Install Visual Studio Code

## Execution 
In order to run this code, please run the command below in the tests folder:
```bash
python -m unittest discover
```

In order to run the code of one specific file, please run the command below, just changing the name of the p√Ωthon file you want to run:
